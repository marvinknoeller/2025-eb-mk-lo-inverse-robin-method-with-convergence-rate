## Content of this project

This is a guide to generate the figures that have been used in the work

**„A Computational Method for the Inverse Robin Problem with Convergence Rate“**

_By Erik Burman, Marvin Knöller and Lauri Oksanen._

You find all the needed Python and Matlab files in the provided project.

## An overview
Our work features 5 numerical examples. The numerical example j can be reproduced by running 'Example'+j+*'.py'.
Subsequently, the Matlab script 'Example'+j+*'.m' can be run. The Matlab scripts are used for plotting only. 

# A short explanation of the scripts

All **.py** files hold descriptions. In particular, all Python defs are described in detail.

The folder precomputed_files includes all the precomputed results that we computed in order to generate Figures 2-6.
These are the files that are generated by running 'Example'+j+*'.py' for j = 1,...,5

## Requirements and additional information
The Python scripts have been computed with the following (most relevant) packages and versions:
- numpy: 2.2.6
- dolfinx: 0.9.0 (see in particular https://fenicsproject.org)
- ufl: 2024.2.0
- petsc4py: 3.23.2
